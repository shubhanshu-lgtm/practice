
1.2 Service Details – Form Specification
Section Purpose
Capture what services the lead is interested in, who owns the lead, current status, and structured follow-up history — all tied to a single enquiry.

A. Enquiry Information
1. Enquiry ID
Field Type: Dropdown (Searchable)
Mandatory: ✅ Yes
Source: search from Lead Details (1.1)
Format: IS/YY/MM/DD/01
Behavior:
Dropdown shows only active enquiries
On selection → auto-populate:
Enquiry Date
Customer Name (read-only, optional display)
Editable: ❌ No
Validation: Must exist in Lead master

2. Enquiry Date
Field Type: Date Picker
Mandatory: ✅ Yes
Default Value: Auto-filled from Enquiry creation date
Editable: ❌ No (recommended)
Format: DD-MM-YYYY

B. Service Requirement Details
3. Service Requirement
Field Type: Multi-Select Dropdown + Add Custom Option
Mandatory: ✅ Yes
Source: Service Catalogue Master
Selectable Options Example:
ISO 27001 Certification
SOC 2
VAPT
PCI DSS
Training
Cloud Security
Features:
Can select multiple services
Searchable dropdown
➕ Add Custom Service.
Custom Service Fields (Conditional):
Custom Service Name (Text, Mandatory)
Description (Textarea, Optional)
Validation:
At least one service required
No duplicate services allowed

4. Service Department
Field Type: Dropdown (Single Select)
Mandatory: ✅ Yes
Options:
GRC
VAPT
Training
PCI
MSC Noida
MSC BLR
Behavior:
Auto-suggest based on selected service (optional)
Manual override allowed

C. Ownership & Status
5. Lead Owner
Field Type: Dropdown (Searchable)
Mandatory: ✅ Yes
Source: Sales Personnel Master / System Users
Display Format: Name – Employee ID
Default Value: Logged-in user (configurable)
Editable: ✅ Yes (based on role)
Validation: Must be an active user

6. Lead Status
Field Type: Dropdown
Mandatory: ✅ Yes
Options (Lifecycle-Based):
New Lead
Contacted
Information Collection
Proposal Submitted
Awarded
Lost
Behavior Rules:
Status flow can be enforced (optional)
If Lost → show Lost Reason
If Awarded → trigger Opportunity / Project creation
Lost Reason (Conditional)
Field Type: Dropdown + Remarks
Options:
Budget Constraints
Lost to Competitor
No Response
Requirement Cancelled
Others

F. UI/UX Recommendations

G. Validation Rules Summary
❌ Cannot submit without Enquiry ID
❌ At least one service required
❌ Lead Owner mandatory
❌ Status mandatory
❌ Lost → Reason required.
