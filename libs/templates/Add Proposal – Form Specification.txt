2. Add Proposal – Form Specification
Section Purpose
Create and submit a commercial proposal linked to an enquiry and its selected services, with line-level pricing, tax handling, payment milestones, and document management.

A. Proposal Header Information
1. Enquiry ID
Field Type: Dropdown (Searchable)
Mandatory: ✅ Yes
Source: Lead / Service Details
Behavior:
On selection, auto-populate:
Customer Name (read-only)
Selected Services (from Section 1.2)
Editable: ❌ No

2. Proposal Reference
Field Type: Text (Read-only)
Auto-Generated: ✅ Yes
Recommended Format:
PROP/{EnquiryID}/{Seq}

Example:
PROP/IS/26/01/15/01/01

📌 Reason: Supports multiple proposals per enquiry (revision handling)

3. Proposal Date
Field Type: Date Picker
Mandatory: ✅ Yes
Default: Current Date
Editable: ✅ Yes (with permission)

4. Proposal Submitted By
Field Type: Dropdown
Mandatory: ✅ Yes
Source: Location / Legal Entity Master
Options:
INTERCERT INC
INTERCERT NOIDA
INTERCERT BLR
INTERCERT SOUTH AFRICA
Recommendation: Store as Legal Entity ID, not text

9. Upload Proposal
Field Type: File Upload
Mandatory: ✅ Yes
Allowed Formats: PDF, DOCX
Max Size: Configurable (e.g. 10 MB)

10. Upload Supporting Documents
Field Type: Repeatable Upload
Fields:

File Upload
Mandatory: ❌ Optional

F. Notes & Internal Comments
11. Notes
Field Type: Textarea
Mandatory: ❌ Optional
Max Length: 500 characters
Purpose: Internal remarks only (not client-facing)

G. Status Automation (CRITICAL)
12. Lead Status Update
System Rule (Mandatory):
On successful proposal save & upload:
Lead Status → PROPOSAL SUBMITTED

User Override: ❌ Not allowed
Audit Log: Required

H. Recommended Enhancements (My Suggestions)
✔ Proposal Versioning (Highly Recommended)
Version: V1, V2, V3
Allow resubmission without overwriting history

✔ Validity / Expiry Date
Proposal Valid Until (Date)
Default: +30 days

✔ Approval Workflow (Optional)
Sales Manager Approval
Finance Approval (if discount > X%)

✔ Multi-Currency Lock
Prevent mixing currencies within a single proposal (recommended)

I. Validation Summary (Final Checklist)
❌ Proposal cannot be submitted without document upload
❌ Service amount must be > 0
❌ Payment terms must total 100%
❌ Lead status auto-change enforced
❌ GST applied only for India-INR
